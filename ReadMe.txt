    Структура Проекту

OUT/
├── main.py
├── config.json
├── api_keys.json
├── wallets.csv
└── utils/
    ├── __init__.py
    ├── balance.py
    ├── fee.py
    ├── gwei.py
    ├── signature.py
    ├── wallet.py
    
        
    Пояснення:
Файл utils/signature.py:

Містить функцію для генерації підпису для API запитів.
Файл utils/balance.py:

Містить функції для перевірки балансу та фільтрації даних балансу.
Файл utils/fee.py:

Містить функцію для перевірки комісії на виведення.
Файл utils/gwei.py:

Містить функції для отримання поточного значення GWEI та перевірки, чи воно менше максимального значення.
Файл utils/wallet.py:

Містить функції для обробки діапазонів індексів гаманців та зчитування адрес з файлу CSV.
Файл main.py:

Основний файл, який імпортує всі функції з інших файлів і виконує основну логіку програми.
